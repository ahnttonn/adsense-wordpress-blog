## Reusable Prompt Set
These prompts are scaffolds, not autopublish buttons. Each one should force the model to cite inputs, separate facts from recommendations, and produce reviewable sections. For blog prompt library, the best template asks for a draft plus an evidence checklist so the editor can reject unsupported claims quickly.

## How to Adapt the Templates
```text
Role: Act as an editorial operator.
Task: Create a concise draft for [topic] using only the provided sources.
Inputs: [source URLs], [audience], [search intent], [constraints].
Output: quick answer, outline, evidence notes, risks, internal links.
Rules: mark uncertainty, avoid copied wording, and list every claim that needs rechecking.
```
