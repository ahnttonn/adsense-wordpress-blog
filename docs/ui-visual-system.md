# UI Visual System

The WordPress theme must follow the Task 17 evidence files:

- `.omo/evidence/task-17-ui-benchmark.md`
- `.omo/evidence/task-17-design-brief.md`
- `.omo/evidence/task-17-wireframes.html`

Direction: restrained editorial/productivity, typography-led, readable, ad-safe, and explicitly not AI-SaaS styled.

Implementation source of truth:
- Warm paper and ink palette with teal/clay accents.
- Serif article typography, sans UI typography, mono code blocks.
- Compact homepage lead plus dense topic rails.
- Article pages with quick answer, table of contents, comparison tables, source notes, update log, and reserved ad slots.
- No purple/blue gradient blobs, glowing orbs, glass cards, robot imagery, fake dashboards, oversized empty heroes, nested cards, or page-builder look.
