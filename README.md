# AdSense WordPress Blog

Project repository for the English-first AI tools and automation WordPress publication.

## Scope
- WordPress theme/configuration notes.
- Infrastructure scripts and deployment runbooks.
- Editorial strategy, content templates, SEO/GEO/AEO checklists, and operations docs.
- No private keys, credentials, WordPress salts, backup archives, or account secrets.

## Operating Playbooks

- [`docs/blog-monetization-verdict.md`](docs/blog-monetization-verdict.md): AdSense-only monetization verdict, current-content limits, Auto ads path, and invalid-traffic posture.
- [`docs/content-pillar-map.md`](docs/content-pillar-map.md): launch inventory reconciliation, 90-day operator-tech pillar mix, and queued briefs.
- [`docs/seo-aeo-geo-guide.md`](docs/seo-aeo-geo-guide.md): SEO, AEO, and GEO gates for answer-first, citation-visible articles.
- [`docs/ad-monetization-playbook.md`](docs/ad-monetization-playbook.md): Auto ads, no live manual units before approval, consent/CMP, and invalid-traffic rules.
- [`docs/hourly-publishing-automation.md`](docs/hourly-publishing-automation.md): guarded hourly draft queue and no blind public auto-publish policy.
- [`docs/analytics-search-reporting.md`](docs/analytics-search-reporting.md): weekly refresh, expand, prune, merge, Auto ads UX, and AdSense blocker decisions.
- [`docs/deployment-rollback-guide.md`](docs/deployment-rollback-guide.md): content, theme, Auto ads, SEO, and hourly scheduler rollback.

## Account-Owned Actions

The repository can prepare evidence and run checks, but the site owner controls AdSense submission, Google account identity, payment and tax setup, Search Console verification, Bing Webmaster Tools verification, AdSense dashboard Auto ads settings, and any account-side consent/CMP configuration.
